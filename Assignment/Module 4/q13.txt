# Explain Exception handling? What is an Error in Python?

Exception Handling in Python is a mechanism to handle errors or exceptional situations that occur during the execution of a program.
This ensures that the program doesn't crash unexpectedly and and work properly in flow.

Error in python:

-In Python, an error is something that goes wrong in your program, stopping it from running properly.

Types of error in python:

1. Syntax Errors: These occur when the code is written incorrectly.
